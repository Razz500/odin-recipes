# odin-recipes
Practice basic html by creating a simple recipe website

This is my first HTML project with The ODin Project. The aim is to 
create a simple website with a homepage (index) that will link to a few
different recipes. The elements of HTML that will be tested will be:

- Basic boilerplate setup
- Headings
- Lists (both unordered and ordered)
- Links (relative unless linking externally)
- Images
